This is a browser version of something between a sketchpad & Etch-A-Sketch. This project is part of the curriculum of [The Odin Project](https://www.theodinproject.com).

Took me almost a month to complete this one. 
**Repo**:- https://github.com/captainnairauk/etch-a-sketch
**Live demo**:-  https://captainnairauk.github.io/etch-a-sketch/

**Challenges faced**:- 
1. CSS was the monster in this one. Especially when I was trying to make this game mobile friendly. The one tip that REALLY helped me was this ***border: 1px solid black;***. This thing is like console.log equivalent of CSS.

2. Was struggling badly with Flex. Flexbox froggy (https://flexboxfroggy.com/) was a major help in making this project responsive. 

3. I had decided to make this project mobile first, which meant I had to understand Media query (Previously I was scared of this topic for some reason and always either avoided it or skipped this part.) I watched a lot of videos based on Media query but I still didn't know how to apply it. Earlier I had faced same issue with DOM manipulation and I learned it by reading a freeCodeCamp blog. In case of media query, again freeCodeCamp came to my help. I applied Media query to my project using this article. (https://www.freecodecamp.org/news/css-media-queries-breakpoints-media-types-standard-resolutions-and-more/)

**Takeaways**:- 
1. You simply don’t know what you don’t know yet. i.e. Earlier I was able to finish the other two previous projects in a set deadline. However, that changed with this project. It really opened my eyes to the fact that I can't really estimate, when I am going to complete the project. Basically this project was a very good exercise in managing my own expectations.
